Папка для исходных текстов корпуса.

Структура:
- `data/raw/bible/*.txt`
- `data/raw/commentaries/*.txt`
- `data/raw/sermons/*.txt`

Рекомендуемый формат файлов:

```text
# title: Название
# author: Автор
# reference: Ссылка/цитата

Основной текст...
```

Важно: добавляйте только тексты, которые можно использовать в учебном проекте с точки зрения авторских прав.
