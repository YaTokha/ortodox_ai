# Техническое задание (ТЗ)

## 1. Наименование
Прототип интеллектуального ассистента для анализа Священного Писания и генерации православных проповедей на основе NLP.

## 2. Цель
Создать веб-сервис, который принимает богословский запрос, анализирует фрагменты Писания и формирует черновик православной проповеди на базе дообученной GPT-2.

## 3. Основание для разработки
ВКР: "Разработка прототипа интеллектуального ассистента для анализа Священного Писания и генерации православных проповедей на основе методов обработки естественного языка (NLP)".

## 4. Состав системы
1. Веб-клиент (HTML/CSS/JS).
2. API-сервер (FastAPI).
3. Модуль retrieval по корпусу православных текстов.
4. Модуль генерации (GPT-2 + LoRA адаптер).
5. Пайплайн подготовки датасета и обучения.

## 5. Входные данные
1. Тексты Священного Писания (Ветхий и Новый Завет).
2. Толкования православных отцов церкви.
3. Проповеди православных авторов.
4. Пользовательский запрос: тема, библейский фрагмент, аудитория, контекст.

## 6. Выходные данные
1. Аналитический ответ по заданному фрагменту.
2. Черновик проповеди с планом.
3. Список релевантных источников из корпуса.

## 7. Нефункциональные требования
1. Ответ API в JSON.
2. Поддержка запуска в CPU- и GPU-режиме.
3. Возможность обучения на удалённом Linux-сервере.
4. Логирование и проверка работоспособности (`/api/health`).

## 8. Ограничения
1. Результаты ИИ носят вспомогательный характер.
2. Обязательна экспертная богословская проверка перед практическим использованием.
3. Корпус должен формироваться с учётом авторских прав.
